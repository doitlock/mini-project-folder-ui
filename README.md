# React ToDo App (Webpack + TypeScript)

A simple ToDo List app built with React, TypeScript, Webpack, and SCSS.

## Features
- Add / remove todos
- Mark as complete
- Filter by status (all / active / completed)
- Type-safe with TypeScript

## Tech Stack
- React
- TypeScript
- Webpack
- SCSS

## Setup
```bash
npm install
npm run start
